## USE
---

### Install global
```
$ npm install -g gulp
$ npm install -g bower
```

### Install Local
```
$ npm install
$ bower install
```
### Development mode
```
$ gulp
```

### Production mode

new folder is created `dist`

```
$ gulp production
$ gulp csscompress
```
